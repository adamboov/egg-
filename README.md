# egg-demo
# Egg框架

## 是基于nodejs和Koa的一个企业及应用开发框架，基于Koa开发性能优异，内置多进程管理。

## 相关环境
  - vant ui框架
  - vue-cli3
  - moment.js

## npm init egg --type=simple

## cnpm i 安装依赖

## cnpm i egg-view-ejs --save 安装模板引擎

### Development

```bash
$ npm i
$ npm run dev

```

### Deploy

```bash
$ npm start
$ npm stop
```
